abstaction atau interface
- skema dari class, hanya menampilkan nama-nama method

nanti di class yang mengimplementasikan interface
- harus menyertakan semua isi dari interfacenya

kegunaan lain:
- interface bisa sebagai "bentuk simpel" dari isi class yang bisa ratusan - ribuan baris
- contoh kasusnya, programmer mau mengetahui, "class ini bisa ngapain aja sih?"
- report jika harus scroll-scroll kodingan, ini salah satu keuntungan interface
- cukup liat interfacenya untuk mendapatkan list method
